Run Control

Authors
David Quarrie, (original eiffel version)
Jie Chen, (C++ version)
Graham Heyes, (touched up a few things)
This is the main control interface for the CODA data acquisition system. Help is availible in several forms via the interface, popup "balloon" help, context sensitive message bars and HTML pages. If further assistance is required please contact the data acquisition group.

org.jlab.coda.et

Class Attachment

  • java.lang.Object
    • org.jlab.coda.et.Attachment
public class Attachment
extends java.lang.Object
This class defines an ET system user's attachment to a station. Attachments can only be created by an ET system's SystemUse.attach(Station) method. Attachments are means of designating the ownership of events and keeping track of events.
